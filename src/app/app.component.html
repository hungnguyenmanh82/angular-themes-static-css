cách 1: change themes by static css file

change static css theme ở runtime ở index.html => easy
Lưu ý: khi deploy product build phải trỏ đúng url css ở server
có thể đổi tên "assets" folder và add nó vào mục "assets" ở angular.json
Mỗi lần thay CSS sẽ download lại CSS mới => cần cache static file

để tránh trong thời gian download new css theme sẽ bị mất theme
ta dùng ajax download css ở background trc (css có cached).
nếu download thành công thì tiến hành đổi theme (nếu fail thì vẫn dùng theme cũ)
